# E-Commerce Cart System (Python)

## Overview
This project is a command-line based e-commerce system developed using object-oriented programming in Python. It simulates real-world shopping functionality including product management, cart operations, and checkout processing.

## Features
- Object-oriented design using inheritance for product categories
- Cart management system (add/remove items)
- Discount and pricing logic
- File-based persistence for storing transactions
- Simulated payment processing and receipt generation

## Tech Stack
- Python
- OOP Concepts (Classes, Inheritance)
- File Handling

## System Design
The system is structured using multiple classes representing products, cart, and transactions. It allows users to interact with a simulated shopping environment and processes orders with stored transaction records.
